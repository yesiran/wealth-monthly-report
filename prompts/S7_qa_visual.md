# S7 成品质检

你在月报流水线的 S7 步骤，对刚生成的 PDF 做出厂质检。

## 操作步骤

1. 用 Read 工具分批读取 `{PERIOD_DIR}/08_report.pdf` 的全部页面（每次 ≤4 页）。
2. 对照下面的检查清单逐项核查。
3. 写出 `{PERIOD_DIR}/05_notes/qa_report.json`。

## 检查清单

**排版类**
- [ ] 无未解析的 `{{token}}` 或 `%%` 字样出现在页面上
- [ ] 图表标签无重叠/截断（尤其条形图左侧长名称、数值标签与名称相撞）
- [ ] 无孤行标题（章节标题悬在页底、内容在下一页）
- [ ] 表格未被截断、每页页脚正常
- [ ] 每个标的卡片有迷你走势图，月中买入的标的有金色买入标记

**数据一致性类**（对照 `{PERIOD_DIR}/03_returns.json`）
- [ ] 封面组合盈亏/收益率 = summary 数值
- [ ] 收益条形图的排序与数值和 holdings 一致；红=涨绿=跌无颠倒
- [ ] 明细表合计行与 summary 一致

**内容类**（抽查）
- [ ] 随机抽 3 个标的：时间线日期都落在报告月内；抽 2 个研究性数字回查 `04_research/` 笔记里有出处
- [ ] 每个标的的预判存档都有验证窗口
- [ ] 观察日历日期升序、无重复

## 输出格式

```json
{"pass": true,
 "issues": [{"severity":"blocker|minor","where":"第3页 收益图","desc":"具体问题"}],
 "checked_pages": 16}
```

blocker = 数字错误/未解析占位符/图表严重错乱（必须修复重新出版）；minor = 美观类瑕疵（记录即可）。

完成后只输出一行：`S7 done: pass=true/false, blocker N, minor M`。
